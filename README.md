# Customer Churn Prediction

Machine Learning project to predict whether a customer will leave a telecom service.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Workflow
1. Data preprocessing
2. Exploratory data analysis
3. Feature engineering
4. Training classification models
5. Evaluating model performance

## Dataset
Telco Customer Churn dataset.

## Goal
Predict customer churn using machine learning models to help companies retain customers.
